# 006 Collection Detail

## Overview

Surface game details in the collection view, using the GameDetails component.
When a user clicks on a game in their collection, a modal will open showing the details of the 
game and all available extension actions.

## Required Functionality

 - Modal is full screen on mobile and a centered dialog on desktop.
 - Modal can be closed by clicking outside of it or by clicking a close button.
 - Modal shows a larger 400x400 image of the game (constrained by the width of the modal / screen)
 - Collection grid/list item includes a direct link to the BGG page and collection view for the game, which opens in a new tab.
